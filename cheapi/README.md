### 车牌归属地查询

关键词返回归属地,可能添加的关键词不全,需自行加入

 调用方法 hepai.php?msg= 

### 实例 

```
https://www.0735.pro/api/chepai.php?msg=湘A
```
### 返回值 

```
{
    "code": 1,
    "msg": "获取成功!",
    "data": {
        "province": "湖南",
        "city": "长沙",
        "abbreviation": "湘A"
    }
}
```
